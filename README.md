## 목차
- [개요](#개요)
- [기능](#기능)

##개요
- 시멘틱한 HTML을 구현합니다.
- 확장성을 고려하여 모듈화를 진행하였습니다. 

##배포
- vercel을 사용한 배포

##기능
- 주소값을 받아 좌표값을 받아오는 api 
- 좌표값을 기반으로 지도와, 마커를 생성
- 스크롤 이벤트 처리
